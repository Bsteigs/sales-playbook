# Sales Playbook — Abacus Group

A live, interactive sales playbook built as a web application, designed to be referenced in real time during discovery calls, mid-cycle conversations, and closing sequences.

## What It Does

- Full discovery, mid-cycle, and closing frameworks organized by stage
- The Only 12 Questions pinned at the top of every discovery session
- MEDDPICC, Challenger, and SPIN frameworks accessible in one click
- Field rules and deal killers always one click away
- Password-protected inline edit mode — click any question to update it, changes save instantly
- Persistent storage via Supabase — edits sync across all devices in real time

## Tech Stack

- **HTML/CSS/JavaScript** — single-file web app, no framework needed
- **Supabase** — PostgreSQL database for persistent content storage
- **GitHub Pages** — free hosting, live at a public URL

## Background

Built as a replacement for a Notion-based sales process doc that wasn't usable during live calls. The goal was a tool optimized for a second monitor — fast, scannable, and editable without leaving the browser. Incorporates frameworks from MEDDPICC, Challenger Sale, SPIN Selling, Brian LaManna, Jen Allen-Knuth, and 30 Minutes to President's Club.
